# tools
simple command line tools

## jsonsmd5
```
usage: jsonsmd5.py [-h] [-p PATTERN]

Compute md5 of normalized jsons file

optional arguments:
  -h, --help            show this help message and exit
  -p PATTERN, --pattern PATTERN
                        glob pattern of jsons files
```

## random_pw
```
usage: random_pw.py [-h] [-N N] [--no_specials]

Generate N character random password (numbers, letters, specials)

optional arguments:
  -h, --help     show this help message and exit
  -N N
  --no_specials
```
